# Article-Spider 改造日志

## 项目背景

基于 [Anning01/article-spider](https://github.com/Anning01/article-spider) 改造，原项目是一个 Python 异步新闻爬虫，支持 8 个中文新闻源（网易、搜狐、新浪、澎湃、IT之家、腾讯新闻、腾讯体育、中国日报），数据存入 PostgreSQL。

## 改造目标

将通用新闻爬虫改造为**医药行业热点爬虫**，为安徽省独家代理业务提供行业资讯监控。

### 业务信息

- **Meditrusthealth（康联达健康）**：康哲药业旗下，药品引进/开发/CDMO/营销，覆盖肿瘤、中枢神经、自免、眼科等领域
- **imsrmt.com（曼斯普医学科技）**：医疗影像，主营 MRI / PET/MRI 多模态成像系统
- **角色**：两公司在**安徽省的独家代理**
- **需求**：监控医药行业热点、安徽省医疗政策、药品审批动态、医疗器械行业动态

### 改造思路

1. **关键词过滤**：新增 `keywords.py`，配置业务相关关键词，在入库前过滤只保留相关文章
2. **新增医药爬虫源**：
   - 行业媒体（赛柏蓝、健识局、医药魔方等）
   - 政策机构（国家药监局、安徽省卫健委等）
   - 医学资讯（医脉通、丁香园等）
3. **热点聚合**（可选）：同主题聚类，每日生成行业简报

## 操作记录

### 2026-06-22：完整改造实施（Superpowers 全流程）

**策略调整**：放弃"先过滤后加源"的原始计划，改为**先加医药源、再注册、再加过滤**，确保系统跑起来立刻有产出。

#### 变更文件清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `keywords.py` | 新建 | 4 类关键词（公司/业务/区域/行业），含简短英文词边界防误匹配 |
| `nmpa.py` | 新建 | 国家药监局爬虫，3 个栏目（药品公告、公告通告、政策解读） |
| `anhui_wjw.py` | 新建 | 安徽省卫健委爬虫，2 个栏目（工作动态、通知公告） |
| `base.py` | 修改 | 新增 `keyword_filter` 属性 + `_matches_keywords()` 过滤钩子 |
| `main.py` | 修改 | 注册 2 个新爬虫，import NMPASpider + AnhuiWJW |
| `.env.example` | 修改 | 新增 NMPA_INTERVAL / ANHUI_WJW_INTERVAL（600s） |
| `README.md` | 修改 | 更新为医药爬虫标题与原创作声明 |

#### 修复的 Bug

1. **`DEFAULT_LIMIT = int(None)`** — 当 `.env` 未设置该变量时抛 TypeError，改为安全取值
2. **日期格式崩溃** — 列表页无日期时的 `strftime("%Y-%m-%d")` 与 `save_article` 期望的 `%Y-%m-%d %H:%M:%S` 不匹配
3. **PET 误匹配** — 全小写 "pet"（宠物）通过关键词过滤，改为大小写敏感正则 `\bPET\b`

#### 使用方式

```bash
# 配置 .env 后
uv run python main.py
# 独立测试某个爬虫
uv run python nmpa.py
uv run python anhui_wjw.py
```

#### 后续可扩展

- 新增医药媒体源（赛柏蓝、丁香园、医脉通等）—— 按 `xxx.py` + `main.py` 注册模式
- 每日行业简报（可选）—— 从 DB 查询当日入库的医药文章，生成 Markdown 汇总

## 重启后操作指引

重启后对 Claude 说：
> "改造 article-spider，按 CHANGELOG.md 中的改造目标进行"

Superpowers 会自动触发 brainstorming 流程，将按以下步骤推进：

### Step 1：创建 keywords.py
关键词配置文件，包含：
- 公司关键词：康哲药业、康联达、Meditrusthealth、曼斯普、Unitedwell、MRI、PET/MRI
- 业务领域关键词：肿瘤、中枢神经、自免、眼科、药品审批、集采、医保、医疗器械
- 区域关键词：安徽、安徽省、合肥
- 行业关键词：医药、医疗健康、NMPA、CDMO、临床试验

### Step 2：改造 base.py
- 在 `crawl_and_save()` 中加入关键词过滤逻辑
- 标题或内容匹配到任一关键词则保留，否则跳过
- 可配置匹配模式（精确/模糊/权重）

### Step 3：新增医药爬虫源
优先添加（按可爬取难度排序）：
1. 国家药监局公告 - 政策/审批动态
2. 安徽省卫健委 - 省内政策
3. 赛柏蓝/医药行业媒体 - 行业资讯
4. 丁香园/医脉通 - 医学前沿

### Step 4：注册到 main.py
- 将新爬虫加入 `spiders` 字典
- 配置独立的抓取间隔

### Step 5：添加热点报表（可选）
- 每天/每周自动汇总相关的热点文章
- 输出 Markdown 格式的行业简报
